🗺️ Mi Ruta de Aprendizaje Frontend / Backend

⏱️ Tu ritmo: 2h/día (Lun - Vie)
👉 ~10 horas por semana
👉 ~40 horas por mes

🟢 FASE 1 — JavaScript Sólido

3 Meses

🎯 Objetivo: dejar de "pensar en código" y empezar a entenderlo.

  • Variables, tipos
  • Funciones (normal/flecha)
  • Arrays (map, filter, reduce)
  • Objetos (MUY importante)
  • Scope básico
  • DOM (manipular HTML)
  • Eventos
  • async/await + fetch (APIs)

🔥 Proyecto: App de tareas (con localStorage)

🧠 Reto clave: consumir una API (ej: usuarios) y mostrar datos en pantalla.

🔵 FASE 2 — React

2 Meses

🎯 Objetivo: aprender a construir interfaces modernas.

  • Componentes
  • Props
  • useState
  • Eventos
  • Renderizado de listas
  • useEffect (básico)

🔥 Proyectos:

  • Contador
  • Lista de tareas en React
  • App que consuma API

🧠 Reto clave: Dividir tu app en componentes.

🟣 FASE 3 — Next.js

2 Meses

🎯 Objetivo: entrar al mundo profesional.

  • Routing (app router)
  • Layouts
  • Fetch de datos
  • API routes (backend)
  • Server vs Client

🔥 Proyecto fuerte (IMPORTANTE):

🔐 App con: login + base de datos + CRUD

👉 ¡Aquí ya estás cerca de ser junior!

🗄️ FASE 4 — Base de datos

2 Meses

🎯 Objetivo: entender datos de verdad.

MongoDB (primero)

  • CRUD
  • Conexión con Next

SQL (después)

  • SELECT, INSERT, UPDATE
  • Relaciones

🔥 Proyecto:

Sistema de usuarios con BD.

🧠 FASE 5 — Backend extra

1.5 Meses

🎯 Objetivo: entender backend de verdad con FastAPI.

  • Crear API con FastAPI
  • Rutas (GET, POST)
  • Conectar con BD
  • Autenticación básica

🔥 Proyecto:

API de usuarios (login + CRUD)

👉 Aquí entiendes frontend + backend separado.

🛠️ FASE 6 — Herramientas reales

1 Mes

🎯 Objetivo: ser "empleable".

  • Git (MUY importante)
  • GitHub
  • Deploy (Vercel, etc.)
  • Variables de entorno
🔥 Proyecto: Subir tu app a internet

🟡 FASE 7 — TypeScript

1.5 Meses

🎯 Objetivo: nivel profesional.

  • Tipos básicos
  • Interfaces
  • Usar TS en React/Next

🔥 FASE FINAL — PROYECTOS REALES

2 - 3 Meses

🚀 Aquí te conviertes en junior REAL

💼 Proyectos Obligatorios (Elige y construye):

1. App Completa

  • Login
  • Base de datos
  • CRUD completo

2. App tipo Producto

  • E-commerce básico
  • - o -
  • Red social simple

3. API Propia

  • Backend propio en FastAPI
  • O usar Next.js Backend
  • Conectado a tu frontend
